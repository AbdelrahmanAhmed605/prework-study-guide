HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Use proper semantic tags like section rather than div (when possible)
  • The link element allows you to import external files to your html file

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Box-shadow will add a shadow to a border with an offset x and y value
  • Percentages can be used for height ONLY if the height of the parent is specified

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • It is good practice to pull origin main before you push your changes to ensure your clone repository is up to date
  • To update your git, you need to add, commit, push, then check your pull request to merge

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements allow us to deter from the natural control flow that is executed by our code.
